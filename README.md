# Visual Studio Code - Extension - F4lk0n FS Explorer
This project aims to create a extension for the VSCode IDE with a enhanced File System Explorer.



## Requirements
- Node.js: 14.x or higher




## Development
Project Start:
```shell
npm install -g yo generator-code
```
